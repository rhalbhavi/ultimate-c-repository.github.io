# Variables in C

Variables are named memory locations used to store data during program execution. In C, every variable must be declared with a specific data type before it can be used.

**Key concepts:**
- Variable declaration: `int age;`
- Variable initialization: `int age = 25;`
- Fundamental data types: `int`, `float`, `char`, `double`, `void`
- Naming rules and conventions
- Scope and lifetime of variables
