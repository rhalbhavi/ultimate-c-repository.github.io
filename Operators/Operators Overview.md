# Operators in C

C provides a rich set of operators for performing various operations on data. Operators are symbols that tell the compiler to perform specific mathematical, relational, or logical operations.

**Types of operators:**
- **Arithmetic**: `+`, `-`, `*`, `/`, `%`
- **Relational**: `==`, `!=`, `>`, `<`, `>=`, `<=`
- **Logical**: `&&`, `||`, `!`
- **Bitwise**: `&`, `|`, `^`, `~`, `<<`, `>>`
- **Assignment**: `=`, `+=`, `-=`, `*=`, `/=`, `%=`
- **Conditional (Ternary)**: `?:`
- **Special**: `sizeof()`, `&` (address-of), `*` (dereference), `,` (comma)
- **Increment/Decrement**: `++`, `--`
