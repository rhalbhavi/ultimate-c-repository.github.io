# Type Conversion in C

Type conversion (type casting) allows converting a value from one data type to another in C.

**Types of conversion:**
- **Implicit Conversion**: Automatically performed by the compiler when mixing data types (e.g., `int` to `float`). Follows type promotion rules.
- **Explicit Conversion**: Manually specified by the programmer using the cast operator `(type)`. Also known as type casting.
