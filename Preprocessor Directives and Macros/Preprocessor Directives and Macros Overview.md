# Preprocessor Directives and Macros in C

The C preprocessor processes directives before compilation begins. These directives begin with `#` and are handled by the preprocessor, not the compiler.

**Key directives:**
- `#define` — Define symbolic constants and macros
- `#include` — Insert contents of another file
- `#if`, `#ifdef`, `#ifndef`, `#else`, `#elif`, `#endif` — Conditional compilation
- **Parameterized Macros** — Macros that accept arguments (function-like)
- **Nested Macros** — Macros defined using other macros
- **X-Macros** — Advanced technique for maintaining lists and generating repetitive code
